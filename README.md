# COVID-19 Misinformation Analysis

A Jupyter Notebook project analyzing COVID-19 misinformation through computational text analysis. Made with Sam Rigor at BU for CAS LX390: Metrics and Evaluation in NLP under Dr. Najoung Kim.

## Overview

This project explores patterns in COVID-19 misinformation using text-analysis methods. It was originally developed for an LX390 final assignment and has been cleaned into a portfolio-style research notebook.

The notebook demonstrates how computational linguistics and NLP methods can be used to examine public-health discourse, misinformation framing, and language patterns in pandemic-related text.

## Setup
Install dependencies:
```bash
pip install -r requirements.txt
```
Then open the notebook:
```bash
jupyter notebook notebooks/covid19_misinformation_analysis.ipynb
```

## Notes
This project is exploratory and educational. It should not be interpreted as medical advice or as a definitive misinformation detection system.
